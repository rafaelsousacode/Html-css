Formatações adicionais

Marcação

Parte dessa frase está marcada usando a tag MARK.

Se você quiser que todas as marcações da mesma página sejam da mesma cor, basta criar um seletor no head do código, olha lá.

Texto grande e pequeno

Nesta frase temos uma palavra grande e uma Pequena

Para aumentar usamos a tag BIG que está em desuso.
Já para diminuir usamos tag SMALL.

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido mas não considerado. através da tag DEL.

Texto inserido

Podemos marcar um texto como inserido para dar ênfase que ele foi adicionado através da tag INS.(semântica)

Também pode se usar a tag U para sublinhar um texto.(forma)

Texto sobrescrito

Usado para inserir coisa do tipo x20+3 Usando a tag SUP (pra cima)

texto subscrito

Usado para escrever coisa do tipo H2O usando a tag SUB (pra baixo)
